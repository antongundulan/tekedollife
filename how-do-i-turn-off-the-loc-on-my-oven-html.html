What does the f90 symbol on a stove mean?

The F90 code indicates that a fault has been detected with the door lock mechanism by the electronic oven control board. When you attempt to initiate a self-clean cycle that makes use of the door lock, the F90 error code may occur.

What does the code f5e1 on a Whirlpool oven mean?

The diagnostic code F5E1 indicates that the over door latch is not functioning properly. As a first step, I propose that you create a clean cycle and run it to see whether the door latches. If the door latches and the code is shown, the problem is most likely with the control; however, if the door does not latch at all, the problem is most likely with the latch mechanism.
